# Beemafy 🏥
> **Automated Health-Link Portal for Insurance Claim Verification**

Beemafy is a Java-based application designed to bridge the gap between patients, hospitals and insurance providers. It automates the verification process, ensuring that claim handling is faster, more transparent, and less prone to manual errors.

## 🚀 Features
- **Interactive GUI:** Built with **JavaFX** for a seamless and user-friendly experience.
- **Automated Verification:** Core logic implemented to cross-reference hospital data with insurance policies.
- **Robust File Handling:** Efficient data management using persistent file-based storage (TXT/CSV) for patient records and claim history.
- **Project Management:** Organized using **Maven** for clean dependency management and scalability.
- **Secure Processing:** Logic-based validation to ensure only legitimate claims are processed.

## 🛠️ Tech Stack
- **Language:** Java
- **Framework:** JavaFX (for UI/UX)
- **Build Tool:** Maven
- **Storage:** File Handling (Input/Output Streams)

## 📂 Project Structure
- `src/main/java`: Contains the core logic and controller classes.
- `src/main/resources`: FXML files for the UI and CSS for styling.
- File Storage: Data is stored in persistent .txt files within the project directory.
